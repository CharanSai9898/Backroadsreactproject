 About Website
A  travel website built using **React** that showcases popular tours, featured services, and travel destinations with reusable components and dynamic data rendering.

## Features

- Responsive navigation bar
- Hero landing section
- About section
- Featured services
- Featured tours
- Social media links
- Dynamic rendering using `map()`
